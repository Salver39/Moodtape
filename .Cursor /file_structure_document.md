## Папки и их назначение

| Папка              | Назначение                                                              |
|--------------------|-------------------------------------------------------------------------|
| `moodtape_core`    | Основная логика: GPT-парсинг, генерация плейлистов, API-интеграции     |
| `bot`              | Код Telegram-бота, включая обработчики, состояния, локализацию         |
| `data`             | Хранилище SQLite, логов, возможных дампов пользовательских запросов     |
| `auth`             | Работа с авторизацией Spotify и Apple Music                            |
| `utils`            | Утилиты: обработка текста, логика фильтрации, вспомогательные функции  |
| `config`           | Переменные окружения, настройки, мультиязычные словари                 |
| `tests`            | Тесты для ключевых модулей                                             |


## Главные файлы

| Файл                                 | Назначение                                                                 |
|--------------------------------------|----------------------------------------------------------------------------|
| `main.py`                            | Точка входа. Запускает Telegram-бот и связывает модули                     |
| `gpt_parser.py`                      | Преобразует текст в структуру параметров для подбора музыки               |
| `playlist_builder.py`               | Логика генерации плейлиста по данным от GPT и вкусам пользователя         |
| `telegram_bot.py`                   | Основной Telegram-интерфейс: команды, состояние, сообщения                |
| `feedback_storage.py`               | Сохранение и обработка обратной связи от пользователей                     |
| `auth_spotify.py`                   | Получение токена доступа Spotify и извлечение лайков                       |
| `auth_apple.py`                     | Работа с dev-токеном и Apple Music API                                     |
| `locales.py`                        | Хранение мультиязычных словарей                                           |
| `.env.example`                      | Пример переменных окружения                                               |


## Генерируй новые файлы здесь

- `moodtape_core/`
- `bot/`
